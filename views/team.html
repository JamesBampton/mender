PORTFOLIO

Some of the products we have developed


nextGEMS
Application

Overview

Web Portal

Ability to view network paths to key applications and identify potential issues along each step of the path. Providing a high volume of information from multiple sources and combining this information to improve insight into the root cause of the issue.

nextGEMS
Application

Pathtrace
Application

Overview

Web Portal

Uses Layer 4 Traceroute using the TCP protocol with manipulated TTL fields. With LFT we get an End to end measurement of the Path through the network. This includes ASN numbers, hostnames and response times. This data gathering of information is stored and used to feed into Pathview and nextGEMS for viewing.

Pathrace
Application

Pathview
Application

Overview

Web Portal

Today, all device related Data is spread across different tools (FCAPS, Incident, Change tickets. etc.). Pathview brings all this data into one place to make it easy to search and view potential issues.

Pathview
Application

Logan
Application

Overview

Web Portal

Application that centralises logging, providing indepth log analysis through a single web GUI. Provides support for different device types (syslog, QIP, Cisco (ACS, FWSM, PIX/ASA, IOS), Juniper, Checkpoint, F5, Riverbed, TACACS/Radius, LDAP, Citrix NetScaler).
